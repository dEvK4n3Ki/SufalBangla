Please Follow the Instruction For install GoGrocer on your server.

1.) Upload Backend On Your Server

  1. Login to Your Cpanel .
  2. Go to The File Manager And These Folder Where You Want to Upload Your Backend.
  3. Extract Or Unzip  Your Uploaded File. 
  4. Go to the Extracted Folder And Select All Files 
  5. Copy or Move Them to The Previous Folder Where You Want to Keep Them.
  6. After Moving or Paste Your All File to the Main Folder (Where You Want), You Can Delete Zip as Well as The Extracted Folder.
  7. Uploading Session is Done

2.) Start Installation with Our Installer

  1. Go To The URL Where You Have Uploaded Backend like : www.yourdomain.com/folder_name
  2. You Will Get Something Like That. If All Check Click NEXT Button .
  3. Varifying Purchase. here you will have to enter your License Code &  your name which is binge provided by us. If You Will Enter right license code and your name then you will get like this.
  4. Next Step is Database Configuration. but before that you have to create the DATABASE first from your phpmyadmin and save the username, password & database name to somewhere for installation process. THENInsert the database detail like : hostname, database username, database password & database name.
  5. If the whole Process of the installation will be right them you will get this.
  6. Installation session in finish.

3.) How To Configure Push Notification(Firebasr) From Server.

  1. Goto the application folder from cpanel.
  2. Goto helper Folder and open the gcm_helper.php file.
  3. Replace $api_key value with your Server Key .
NOTE : The Server Key You Will Get From https://console.firebase.google.com > Select or Create Your Project > Add App WithYour project > Goto Project Setting > Cloude Messaging 

4. ) How to Configure Instamojo Payment Gateway

  1. INSTAMOJO
  • Go to Server Where Upload Your Backend Script .
  • there is access_tocken.php file 
  • new instamojo (change this value with your instamojo key) .
