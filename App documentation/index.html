2) Installation of Android App

Step 1-File  <  Open Project  <  Select Project
When The Project Is Imported, Run The Project On The Android Device To Test it.
Step 2 . To Change The Logo, please do below. Change The Icon Inside The Drawable Folder Of The App, By Replacing  icon.png  With Own Icon
 Go to Drawable Folder in App > Replace Icon.png with own icon.
Step 3- To Change The Color,
- Go to Resources Folder In App  <  Go to file in Values  <  color.xml  <  Change The Default Color With your Own Color
Step 4- To Change Base Url
****Go to java <  Config <  BaseURL   <  Open File change the BASE_URL strings <  Replace Base Url With your own Base Url.
Step 5- To Change The App Name
- Go to Resources Folder In App  <  Go to File In Values string.xml <  Change the App name with your own App Name.
Step 6- To change Package Name
-Go to Gradle Scripts < build.gradle (Module app)  <  application id <Change with your  own package name.
Step 7- Firebase Push Notification Setup
- Replace the Android with Project  Below the File Name Folder  < Grocery Demo <  app < Replace the google-services.json with your own google-services.json file.
Step 8- To Download google-services file
- Go to Firebase Console Website < Enter the package name of your project < Download the google-services.json file 
Step 9- To Change the Razorpay Payment Gateway Api Key
- Go to AndroidManifest.xml  File  < Go to meta-data Razorapi key < Change the api key with your api key

3.All Set ...Go Live
